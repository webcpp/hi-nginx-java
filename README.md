# hi-nginx-java

a java web framework for hi-nginx:

- QUICK
- EASY
- FAST

## dependence

- jdk 8+

## components

- hi
  - hi.request
  - hi.response
  - hi.route
  - hi.servlet
  - hi.lrucache
  - hi.controller
- mustache template engine
  - [jmustache](http://github.com/samskivert/jmustache)
- config
  - [config](https://github.com/lightbend/config)

  

## install

`make && sudo make install`
